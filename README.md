# CareerStrand – WorkDNA AI Platform

## Overview
This project was developed as part of the PIDEV – 2nd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

CareerStrand is a web platform that helps users discover their digital work potential through a WorkDNA system. It analyzes skills, generates a personalized profile, and recommends suitable opportunities and career paths.

---

## Features
- User registration and profile management
- Skill selection and questionnaire system
- WorkDNA generation (skill analysis)
- Personalized recommendations
- Opportunity matching system
- Application workflow (pending / accepted / rejected)
- Progress tracking and feedback system

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- PHP
- MySQL

---

## Architecture
The system is divided into 5 main modules:
1. User & Profile Management
2. Skills & Questionnaire
3. WorkDNA Analysis & Recommendation
4. Opportunities & Applications
5. Progress & Feedback
